# Elegant-Stowatch
Elegant Stopwatch is a minimalist, web-based timer that measures elapsed time with millisecond precision. It uses a monotonic clock for accuracy across pauses and tab switches, and presents a simple UI with keyboard-friendly controls. Ideal for learning DOM manipulation, timing APIs, and clean event handling.

**Key features**

* Start, stop (pause), and reset controls
* Millisecond precision display: mm:ss.mmm
* Accurate timekeeping using a monotonic clock
* Simple, responsive UI
* Clean, beginner-friendly code structure

**Live demo**

**How to run**
Clone the repo and open index.html in any modern browser
Or visit the live demo link(https://ppriyanshuu7.github.io/Elegant-Stowatch/)
